#  Spotify Library Manager (Python)

Este projeto integra a **API do Spotify** para gerenciar e automatizar o download de bibliotecas musicais extensas (4.000+ itens).

## 🛠️ Habilidades Técnicas Demonstradas
- **Integração de API:** Uso da biblioteca `spotipy` para autenticação OAuth 2.0.
- **Data Handling:** Processamento de metadados de faixas, artistas e álbuns em formato JSON.
- **Automação de Arquivos:** Lógica para organização de diretórios e verificação de integridade de downloads.

## 🚀 Como funciona
O script solicita permissão via Spotify Dashboard, mapeia as faixas da playlist do usuário e gera uma fila de processamento para download e organização local.
